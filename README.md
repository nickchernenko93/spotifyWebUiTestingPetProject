# Spotify Automation Project
This is simple automation project. It is support  Web UI and API testing.

## Supported browser
    Google Chrome

## Stack of technologies
1. Java13
2. Selenium
3. Selenide
4. Serenity
5. JBehave
6. Page Object design pattern
7. Jenkins CI
8. Allure report
9. Junit5
10. Karate

## Plugins
1. JBehave support
2. Karate

## Running test and generating reports
